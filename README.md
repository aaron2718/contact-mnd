# contact-mnd
A minimalist Dash script to search contacts

Insert your path to the contact text file at the end of line 5.
The file might look like the following example:
```
###  
  
Tim Taylor  
Address: Street 1, Berlin  
Phone: 01234 5678912  
  
###  
  
Sam Smith  
Phone: 09876 5432198 
  
###
```
